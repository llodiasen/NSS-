# Site Web NSS - Nous Sommes la Solution

Site web du mouvement panafricain "Nous Sommes la Solution" (NSS) pour la promotion de l'agroécologie et la souveraineté alimentaire en Afrique de l'Ouest.

## Description

Ce site présente les activités, articles et actualités du mouvement NSS qui regroupe plus de 175 000 agricultrices dans huit pays d'Afrique de l'Ouest.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Unicons (icônes)

## Structure

- `index.html` - Page d'accueil
- `article-*.html` - Pages d'articles individuels
- `blog.html` - Page de blog
- `assets/` - Ressources (images, CSS, JS)

## Déploiement

Le site peut être déployé sur Vercel, Netlify ou tout autre service d'hébergement statique.

## Auteur

Mouvement panafricain "Nous Sommes la Solution" (NSS)
